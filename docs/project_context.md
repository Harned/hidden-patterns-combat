# Project Context (MVP)

Проект: моделирование скрытых закономерностей соревновательной деятельности единоборцев на основе HMM.

Цели MVP:
- загрузка и нормализация Excel-данных (включая многорядные заголовки);
- преобразование бинарных признаков в компактные числовые коды;
- обучение/загрузка HMM;
- декодирование скрытого сценария эпизодов;
- формирование интерпретируемых артефактов: таблицы, профиль состояний, графики, краткий текст.

Принцип проектирования:
- разделение слоев `raw -> feature engineering -> modeling -> interpretation`;
- гипотезы исследования (текущая S1-S2-S3 логика) поддерживаются, но не зашиваются как жесткий единственный сценарий;
- конфигурируемость через `PipelineConfig`.
